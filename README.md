# ImageLabelingManager
====

#### 概要

* 画像認識のために，収集済み画像DBへ学習用のラベル付けを行う機能
- 画像管理機能
- 画像認識補助機能



#### ラベリング機能

* 指定画像1枚に対して，任意のラベル付けを行う
- 画像DBへ直接書き込みせず，画像管理機能によって編集後の作業を保存


#### 画像管理機能

* 画像DBから，画像情報，ラベリング情報などを読込
- 画像DBから，画像情報，ラベリング情報などを保存
- 各種ソート機能，ソート後の整列保存機能
- 画像DBへ画像を追加，画像DBから画像を削除


#### 画像認識補助機能

* 学習用データ，テスト用データ群の自動生成
- 生成データ群の出力



#### プロジェクト

* C++ネイティブコードおよびコンソールアプリケーションのプロジェクト(ImageLabelingManager)
- C#プロジェクトから読み込むためのマネージドDLLプロジェクト(ManagedILM)　ラッパーのみ
- Pythonから読み込むためのBoost.Pythonを使うプロジェクト(PythonILM)　ラッパーのみ
